## Getting started

### Prerequisites

You'll need NPM to start with
* npm
  ```sh
  npm install npm@latest -g
  ```
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/louannl/unitakeaway.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
